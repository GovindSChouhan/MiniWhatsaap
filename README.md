## 📌 About

A simple **Mini WhatsApp CRUD Application** built to learn how **MongoDB**, **Express.js**, **EJS**, and **Mongoose** work together.

This project helped me understand:
- CRUD Operations
- Express Routing
- MongoDB with Mongoose
- EJS Templating
- HTML Forms
- Method Override
- MVC Folder Structure (Basic)

---

## 🚀 Features

- 📄 View all chats
- ➕ Create a new chat
- ✏️ Edit an existing chat
- 🗑️ Delete a chat
- 📅 Display message date and time

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Go inside the project

```bash
cd MiniWhatsApp
```

Install dependencies

```bash
npm install
```

Start MongoDB server.

Run the project

```bash
nodemon index.js
```

or

```bash
node index.js
```

Open in browser

```
http://localhost:8080/chats
```
